# Node JS ES6


## Initialize nodejs project

```shell
npm init -y
```

## Install ES6 and nodemon dependencies

NPM

```shell
npm i -D babel-cli babel-preset-es2015 nodemon
```

YARN

```shell
yarn add -D babel-cli babel-preset-es2015 nodemon
```

## Run dev server

```shell
npm run dev
yarn run dev
```
